## 二、导航

| 命令 | 作用 |
|------|------|
| `pwd` | 打印当前所在目录（print working directory） |
| `cd ~` | 回家目录（~ = /home/yzx 的简写） |
| `ls` | 列出当前目录内容 |
| `ls -l` | 详细信息（权限、大小、时间） |
| `ls -a` | 包括隐藏文件（.开头） |
| `ls -la` | 详细信息 + 隐藏文件 |

### ls -l 输出解读

```
-rw-r--r-- 1 yzx yzx 220 Jun 20 11:11 .bash_logout
└─┬──┘    │ └┬┘ └┬┘ └┬┘ └──┬──────┘ └ 文件名
 权限     链接 属主 属组 大小   修改时间
```

## 三、权限

### 权限字符串拆解（10 个字符）

```
- rwx r-x r-x
│ └┬┘ └┬┘ └┬┘
│  │   │   └── 第三组(3字符)：其他人
│  │   └── 第二组(3字符)：同组用户
│  └── 第一组(3字符)：属主（文件拥有者）
└── 文件类型（- 普通文件，d 目录）
```

**固定规律：去掉第一个字符，剩下 9 位，每 3 个一组。**

```
r = read（读）   w = write（写）   x = execute（执行/进入目录）
```

### 数字权限速算

```
r = 4    w = 2    x = 1     - = 0

三个一组相加：
  r-x = 4+0+1 = 5
  rwx = 4+2+1 = 7
  rw- = 4+2+0 = 6
  r-- = 4+0+0 = 4
  --x = 0+0+1 = 1
```

### 常见权限组合

| 数字 | 权限 | 含义 | 场景 |
|------|------|------|------|
| `600` | rw------- | 只有属主能读写 | 私密配置文件 |
| `644` | rw-r--r-- | 属主读写，其他人只读 | 普通文件默认值 |
| `755` | rwxr-xr-x | 属主全权，别人读+执行 | 可执行脚本 |
| `777` | rwxrwxrwx | 所有人都能全干 | **千万别用，安全灾难** |

### 改权限

```bash
chmod 600 文件名     # 改成只有自己能读写
chmod 755 文件名     # 改成可执行
```

## 四、文件操作

| 命令 | 作用 |
|------|------|
| `mkdir 目录名` | 创建目录（make directory） |
| `cp 源文件 目标` | 复制文件（copy） |
| `mv 源文件 目标` | 移动文件。在同一目录内 = 重命名 |
| `rm 文件名` | 删除文件 |
| `rm -r 目录名` | 删除非空目录（-r = recursive 递归） |
| `rmdir 目录名` | 只删空目录 |
| `cat 文件名` | 看文件全部内容 |
| `head -3 文件名` | 看前 3 行 |
| `tail -3 文件名` | 看后 3 行 |
| `tail -f 文件名` | 实时追踪文件尾部（看日志用），Ctrl+C 退出 |
| `touch 文件名` | 创建空文件，或更新已有文件的修改时间 |

### 写入文件

```bash
echo "内容" > 文件      # 覆盖写入（文件不存在就创建）
echo "内容" >> 文件     # 追加写入
```

## 五、管道

管道 `|` 把左边命令的输出传给右边命令。

### 两个核心模式

```bash
看看有没有 X         →  列出全部 | 搜 X
看看有多少 X         →  列出全部 | 搜 X | 计数
```

### 管道三剑客

| 命令 | 作用 | 什么时候用 |
|------|------|-----------|
| `less` | 分页浏览 | 输出几百行，需要慢慢翻 |
| `grep` | 搜索过滤 | 只要包含关键字的行 |
| `wc -l` | 统计行数 | 想知道有多少个 |

### 实战例子

```bash
ls -la /etc | less              # 慢慢翻看 /etc 目录
ls -la /etc | grep "apt"       # 只看文件名含 apt 的行
ps -ef | grep java              # 找所有 Java 进程
cat 日志.log | grep "ERROR" | wc -l   # 统计日志里有多少 ERROR
du -sh * | sort -h | tail -5    # 找最大的 5 个文件
```

### less 和 grep 的区别

| | `less` | `grep` |
|------|--------|--------|
| 干什么 | 分页浏览，慢慢看 | 过滤，只要匹配的行 |
| 类比 | 翻书 | 用搜索框 Ctrl+F |
| 什么时候 | 不知道具体找什么 | 清楚要找含某关键字的东西 |

## 六、进程管理

### 找进程

```bash
ps -ef           # 列出所有进程（一次性快照，几百行）
ps -ef | grep xxx    # 过滤只找含 xxx 的进程行
```

### top vs ps -ef

| | `top` | `ps -ef` |
|------|------|---------|
| 刷新方式 | **实时**，每 3 秒自动刷新 | **一次性**，输出完就结束 |
| 显示内容 | 进程 + CPU + 内存 + 负载 | 只有进程列表 |
| 交互 | 能排序（P=CPU排序 M=内存排序）、能杀进程（按k） | 不能交互 |
| 什么时候用 | 排查"现在服务器为什么卡" | 配合 grep 找特定进程 |
| 退出 | 按 q | 自动结束 |

**类比：** `top` = 任务管理器（实时面板），`ps -ef` = 拍一张快照。

### 杀进程

```bash
kill PID         # 正常终止（给进程机会保存数据）
kill -9 PID      # 强制终止（立刻死，不保存）
```

```
kill     = 礼貌说"请退出"
kill -9  = 直接拔电源
```

## 七、systemctl — 管理服务

### 只管"注册过的系统服务"

```bash
systemctl status 服务名    # 看状态
systemctl start 服务名     # 启动
systemctl stop 服务名      # 停止
systemctl restart 服务名   # 重启（= stop + start）
```

通过**服务名**操作，不用关心 PID。

### systemctl vs kill

| | `systemctl` | `kill` |
|------|------|------|
| 管什么 | 注册过的系统服务（nginx/mysql/cron） | 任何进程 |
| 找目标 | 通过服务名 | 通过 PID 数字 |
| 什么时候 | 启停正式服务 | 杀临时跑的程序、应急 |

### 常见服务

| 服务名 | 干什么 |
|--------|--------|
| `cron` | 定时任务（每天凌晨备份数据库） |
| `ssh` | 远程登录 |
| `nginx` | Web 服务器 |
| `mysql` | 数据库 |

## 八、sudo — 借管理员权限

```bash
sudo 命令    # 以 root（管理员）身份执行一次
```

| 需要 sudo | 不需要 sudo |
|-----------|-------------|
| 修改 `/etc` 下任何文件 | 操作 `/home/yzx` 下自己的文件 |
| 安装软件（`apt install`） | `ls` `cat` `mkdir` 等日常操作 |
| 启停服务（`systemctl start/stop`） | 创建自己的文件 |
