## 🏆 最终竞争力清单

面试官问你时，你必须能回答的 5 个问题：

1. **你的项目架构是什么？** → Spring Boot 业务层 + Python Agent 层，通过 REST 通信，Docker Compose 部署
2. **Agent 层怎么设计状态管理？** → LangGraph StateGraph + Checkpoint 持久化
3. **RAG 全链路怎么做的？** → PDF → Chunking → Embedding → ChromaDB → 检索 → Rerank → 生成 + 引用
4. **Agent 工具调用怎么保证安全？** → 权限分级 + Docker 沙箱执行
5. **Java 和 Python 怎么通信？** → REST API + token 透传 + 异常处理
